If you use SocNetV in your papers, please cite us as follows:

**Kalamaras, D. (2024). _Social Network Visualizer (SocNetV)_. Social network analysis and visualization software. Home page: https://socnetv.org**

For citing the SocNetV Manual, please use:

**Kalamaras, D. (2024). _The SocNetV Manual_. Social Network Visualizer (SocNetV). https://socnetv.org**
